# ShadowHire AI — AI Candidate Ranking System

## 📌 Project Overview
ShadowHire AI is an intelligent recruitment system that ranks candidates based on job descriptions using AI embeddings.

## 🚀 Features
- Takes job description as input
- Matches candidates using semantic similarity
- Ranks candidates based on relevance score
- Helps recruiters find best-fit candidates quickly

## 🛠 Tech Stack
- Python
- Streamlit
- NLP / Embeddings

## ▶ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run app:
   streamlit run app.py

## 🏆 Purpose
Built for hackathon submission to demonstrate AI-based hiring optimization.
