## 📌 Project Overview

ShadowHire AI is an AI-powered recruitment system that automatically analyzes job descriptions and ranks candidates based on their relevance using semantic similarity and embeddings.

The system helps recruiters quickly identify the best-fit candidates from a large pool of applicants, reducing manual effort and improving hiring efficiency.

---

## 🎯 Problem Statement

Traditional recruitment processes are:
- ❌ Time-consuming
- ❌ Manual and repetitive
- ❌ Difficult to scale for large datasets
- ❌ Prone to inconsistency in evaluation

ShadowHire AI solves this problem using Artificial Intelligence by automating candidate screening and ranking.

---

## ⚙️ How It Works

The system follows a simple AI pipeline:

### 1️⃣ Job Description Input
The recruiter enters a job description with required skills and role details.

### 2️⃣ Candidate Data Processing
Candidate resumes/profiles are converted into structured text format.

### 3️⃣ AI Embedding Generation
Both job descriptions and candidate profiles are converted into numerical vector representations using NLP embeddings.

### 4️⃣ Similarity Matching
The system calculates similarity scores between job descriptions and candidates.

### 5️⃣ Ranking
Candidates are ranked from highest match to lowest match based on AI similarity scores.
